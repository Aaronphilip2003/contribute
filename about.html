    Learn to Contribute to Open-Source!

    Learn More

    Contribute.work is a place for new developers to make their first contribution to open source. As a new developer myself, I had trouble finding an open-source project that I was able to contribute to. I knew there would be others in the same situation and I decided to make something that might be able to help. Since starting this project I've found other similar projects. I highly recommend anyone looking for help to check out first-contributions and first timers only
